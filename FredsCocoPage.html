Fred's Coco Page


Welcome to the first website designed to be directly accessed by a

Color Computer using Drivewire!


This page can be viewed using a very primitive browser that runs under NitrOS9

called Cocobrowser. It consists of 2 parts: A user interface called

Cocobrowser written in Basic09 and a parsing-rendering routine

called htmlparser written in C.
